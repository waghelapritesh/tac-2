# TAC-2 — Think. Architect. Code.

AI-native development CLI with full TUI dashboard.

Based on [GSD-2](https://github.com/gsd-build/gsd-2) (MIT License, by Lex Christopherson).

## Install

```bash
npm install -g tac-2
```

## Usage

```bash
tac                      # Interactive TUI
tac new "idea"           # Full pipeline
tac build "feature"      # Smart build
tac think "idea"         # Explore only
tac go                   # Resume
tac ship                 # PR + review
tac dashboard            # Live TUI
tac status               # Current progress
```
